Console Whisperer Test Page

This page contains buttons that generate various JavaScript errors to test the Console Whisperer extension.

Make sure you have the extension installed and the developer console open (F12) to see the errors and the extension's response.

Reference Error

Generates a reference error by trying to access an undefined variable.

console.log(undefinedVariable);

Type Error

Generates a type error by trying to access a property of null.

const obj = null;
obj.property;

Syntax Error

Generates a syntax error by evaluating invalid JavaScript.

eval('if (true) {');

Range Error

Generates a range error by creating an array with an invalid length.

new Array(-1);

URI Error

Generates a URI error by decoding an invalid URI.

decodeURIComponent('%');

Promise Rejection

Generates an unhandled promise rejection.

new Promise((resolve, reject) => {
    reject(new Error('Promise rejected'));
});

Custom Error

Generates a custom error using console.error().

console.error('This is a custom error message');
